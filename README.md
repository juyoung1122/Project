# 🎬 영화 한 줄 평 관리 프로그램

파이썬의 기본 기능을 활용하여 영화 평점(한 줄 리뷰)을 관리하는 미니 프로그램입니다.

---

## 🔑 주요 기능
* 새 영화 제목과 한 줄 평 입력 및 추가
* 현재 저장된 전체 평점 목록 조회
* 평점 데이터를 `reviews.txt` 파일에 저장
* 저장된 파일에서 평점 데이터 불러오기

---

## 📁 파일 구성 (총 7개 소스 파일)
1.  `main.py`: 프로그램 실행 및 사용자 메뉴 처리
2.  `review_manager.py`: 평점 리스트 관리 핵심 로직
3.  `file_io.py`: 파일 입출력 처리 (저장/불러오기)
4.  `reviews.txt`: 실제 데이터 파일 (초기엔 빈 파일)
5.  `README.md`: 프로젝트 설명 문서
6.  `validator.py`: 사용자 입력 유효성 검사 로직
7.  `review_data.py`: Review 데이터 구조 정의 (데이터 클래스)

---

## 🚀 사용 방법
1.  모든 소스 파일(`.py`)을 같은 폴더에 저장합니다.
2.  터미널이나 명령 프롬프트에서 `python main.py` 명령어를 실행합니다.
3.  메뉴 번호를 선택하여 평점을 관리합니다.
